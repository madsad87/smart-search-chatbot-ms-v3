Smart Search Chatbot Widget Test

This is a test page for the Smart Search Chatbot widget. The chat bubble should appear in the bottom-right corner of the page.

Widget Features

  • Floating Bubble: Click the chat bubble to open/close the widget
  • Persistent Session: Your session ID is maintained across page reloads
  • Keyboard Navigation: Press Escape to close the widget
  • Responsive Design: Try resizing your browser window
  • Accessibility: Screen reader friendly with proper ARIA labels

Implementation

The widget is loaded with this simple script tag:

<script src="dist/ssgc-loader.js" data-config-url="https://example.com/wp-json/ssgc/v1/widget-config" data-color="#007cba" data-position="br" defer></script>
Note: This is a demo version. In a real implementation, the widget would connect to your WordPress site's REST API endpoints for configuration and chat functionality.

Configuration Options

  • data-config-url - URL to your widget configuration endpoint
  • data-color - Bubble background color (hex format)
  • data-position - Position: "br" (bottom-right) or "bl" (bottom-left)
  • data-widget-url - URL to the widget HTML file

Browser Storage

The widget uses localStorage to persist:

  • ssgc:session_id - Your unique session identifier
  • ssgc:ui_open - Whether the widget is currently open

You can inspect these values in your browser's developer tools.
